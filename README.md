# FP&A Budget Variance Analysis Dashboard

## Overview

This project is an interactive Financial Planning & Analysis (FP&A) dashboard built in Microsoft Excel to analyze budget versus actual spending across multiple business departments. The dashboard uses PivotTables, PivotCharts, KPI cards, slicers, and variance analysis to support executive financial reporting.

---

## Dashboard Features

- Budget vs. Actual analysis by department
- Monthly budget vs. actual trend analysis
- Variance analysis by expense category
- Budget allocation by department
- Interactive Month and Department slicers
- Executive KPI cards
- Key business insights
